# 战舰小游戏

一个用 Python 开发的 2D 战舰对战游戏，支持炮台组装、鱼雷攻击、金币系统和防作弊验证。

---

## ✨ 功能特色

| 功能 | 说明 |
|------|------|
| 🎯 多种炮台 | 基础炮、加农炮、导弹架，可自由安装（最多3个） |
| 🚀 鱼雷系统 | 普通鱼雷（3金币）和超级鱼雷（7金币） |
| 🛡️ 防刷币机制 | HMAC-SHA256 签名验证，保护金币系统 |
| 💰 金币经济 | 击沉敌舰获得金币，购买更强装备 |
| 🌊 真实感 | 鱼雷管位于船体水下，水平发射，符合现实 |
| 🔧 IP 账户 | 每个 IP 独立金币池，公平对战 |
| 🌐 一键部署 | 支持 Flask 网页下载 + 本地运行 |

---

## 🚀 如何运行

### 本地运行
- 直接双击 `战舰小游戏.exe`
- 或运行 `python 游戏.py`
- 安装依赖 `pip install flask`
- 访问 `http://<你的IP>:5009`
